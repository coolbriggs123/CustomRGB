# ![ARES](https://i.imgur.com/zjmCumk.png)ARES

A modern, Python-based RGB lighting controller application that integrates with OpenRGB to provide advanced lighting effects and profile management. Built with PyQt6 for a sleek, frameless user interface.

## ‚ú® Features

- **OpenRGB Integration**: Seamlessly connects to OpenRGB SDK to control supported devices.
- **Visual Effect Creator**: Design custom lighting patterns using a layer-based editor (Creator Mode).
- **Profile Management**: Save and load your favorite lighting configurations.
- **Device Control**: View and manage connected RGB devices.
- **Modern UI**: Frameless window design with custom title bar and sidebar navigation.
- **Theming**: Built-in support for Dark and Light themes.
- **System Tray**: Minimized to tray support for background operation.
